What is the MLP Coordinator Hub?
✓

The MLP Coordinator Hub is a set of web tools for managing multilingual/MLP coordinator teams. Everything runs in your browser — no installation, no login required.

Tools included:

  • Classroom Environment Audit — 58-item observation checklist
  • Walkthrough Planning Template — plan rounds, record observations, track coaching
  • Coaching Cycle Tracker — 5-stage coaching pipeline (Observation → Growth)
  • Walkthrough & Audit Dashboards — Chart.js analytics
  • Teacher 360 Profile — aggregate view per teacher
  • Scope & Sequence Tracker — coordinator activity calendar
  • ELPS Knowledge Base — AI-powered ELPS document Q&A
  • Team Overview — manager dashboard across all coordinators

Install as an App (PWA)
✓

Install the suite to your home screen or taskbar for quick access and offline use.

💻
Chrome / Edge (Desktop)
Look for the install icon (⸻) in the address bar, or go to Menu → Install app. The app installs to your desktop.
📱
Android (Chrome)
Tap the 3-dot menu → Add to Home screen. The app icon appears on your home screen.
🍎
iPhone / iPad (Safari)
Tap the Share button ⇧ → Add to Home Screen. Safari must be used (not Chrome on iOS).

Once installed, all tools work offline using cached data.

Sync Your Data to Google Drive
✓

Syncing uploads your local data to a shared Google Drive folder so your manager can see your activity in the Team Overview dashboard.

  1. Open the Data Backup Hub (link below)
  2. Scroll to the Drive Sync section
  3. Confirm your Coordinator ID and Script URL are filled in
  4. Click Push to Drive
  5. You should see "Sync successful" — your manager can now see your data

After syncing, come back and mark this step complete.

Export Walkthroughs to Outlook Calendar
✓

You can export your planned walkthrough rounds directly to Outlook (or Google Calendar) as calendar events.

From the Walkthrough Planning Template:

  1. Open the Walkthrough Planning Template
  2. Fill in the Date / Time Block row for each round (e.g., 2026-03-10 9:15–10:00 AM)
  3. Click the 📅 Calendar button in the toolbar
  4. A .ics file will download — open it to import into Outlook or Google Calendar

From the Coaching Cycle Tracker:

  1. Open the Coaching Cycle Tracker
  2. Click Export Calendar to download coaching deadlines as a .ics file
